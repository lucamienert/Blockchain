# Blockchain in C++

Just trying to create an simple Blockchain from scratch with C++

- Using OpenSSL for sha256 Algorithm
- Using JSONCpp for config file
- Using LibHTTPServer as the Webserice

Things to add:

- p2p
- Webservice

## To build:

```bash
make
```

## To use:

```bash
./blockchain.out <config file>
```
